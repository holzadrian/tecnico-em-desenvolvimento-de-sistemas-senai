1. Imagine que você está montando uma tela de pesquisa paginada (semelhante ao Google), que traz 10 registros de cada vez. A tela lista os filmes do mais velho pro mais recente. Como ficaria o SELECT da página 4?

SELECT * FROM filme
ORDER by ano_lancamento
LIMIT 10 OFFSET 30;

COUNT(*) as qtde

3. SELECIONE de uma vez, em uma só linha, a quantidade de caracteres do MAIOR, do MENOR e também o tamanho MÉDIO de nomes de atores. (Minha base não tem uns campos muito bons pra trabalhar com numeros então tô tendo que inventar isso rs)

SELECT
  MAX(LENGTH(nome)) as maior_nome,
  MIN(LENGTH(nome)) as menor_nome,
  AVG(LENGTH(nome)) as media_nome
 FROM ator;

4. SELECIONE os filmes lançados por ano, ordenando do ano com mais filmes pro ano com menos. 

SELECT titulo_original, titulo_portugues, ano_lancamento, opiniao,
    (SELECT COUNT(*) FROM filme AS f2 WHERE f2.ano_lancamento = f1.ano_lancamento) AS total_filmes
FROM filme AS f1
ORDER BY total_filmes DESC;

5. SELECIONE os anos que temos apenas um filme na base.

SELECT ano_lancamento,
COUNT(*) AS total_filmes
FROM filme
GROUP BY ano_lancamento
HAVING COUNT(*) = 1;
